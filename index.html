About The Project

I started this project because I wanted to challenge myself by coming up with something that would give a final product that was useful for me. The premise of the project is that reading your essay papers backwards (sentence by sentence) helps one catch mistakes in one's writing therefore making the editing process of essay papers easier. The student is able to focus on making each sentence can stand on its own.

Project Requirements

The requirements for this project:

  • The user can enter or paste text on the text box.
  • The text box must accept more than one paragraph.
  • The program must reverse the text that the user enters, reverse the sentences in the paragraph not each word.
  • The user must be able to copy the reversed text, to be able to paste it onto something else.

What I learned

This project taught me how to use my creativity to create something from scratch by building an application that was useful to me as a final product. I wanted to challenge myself to create something which I could use in the future during my studies at Sacramento State.
